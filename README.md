# Dragon Ball Card Game
![dragon ball](img/swiper/dragon.jpg)
DB Card Game is a Dragon Ball Z-themed card game with a variety of features. 
The project is built mainly using **PHP**, **SQL**, **JavaScript**, **HTML**, and **CSS**. Development will continue whenever time permits.

Will try to continue on working on this in my free time if i am able to

## What is DB Card Game?

This card game is based on the Dragon Ball universe. Create your account, open boosters, and start collecting all the cards available in the game. Additional features and functionality will be added over time (hopefully).
This project was developed entirely by myself, from wireframing and mockups to visuals and full development.

## Installation & Start

- Download the project folder.
- Launch it using Go Live (VS Code extension) or any local development server.
- Create/Log in and start playing.

## Features

- Account creation and management
- Booster packs for card collection
- Interactive card gameplay
- Built using PHP, SQL, JS, HTML, and CSS
